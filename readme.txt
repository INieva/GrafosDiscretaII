Proyecto de discreta II - Primera parte

Comandos para compilar el proyecto:

make

Comando para ejecutar el proyecto:

./main

Comando para limpiar los archivos .h y ejecutable:

make clean


Algunas ideas recicladas: Pensar cada vertice como una estructura independiente
en la que almacenamos el nombre, una lista con sus vecinos, el grado (tam de la lista).

IMPORTANTE: Creo que hay un pequenio bug cuando lee una cierta cantidad de vertices
pero el archivo tiene mas vertices.
